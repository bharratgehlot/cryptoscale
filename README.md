# CryptoScale

CryptoScale is a server-side rendered (SSR) cryptocurrency dashboard built with Next.js and Tailwind CSS, focused on clean architecture and SEO-friendly, programmatic pages.

## Tech Stack
- Next.js (Pages Router, SSR)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Project Goal
Demonstrate server-side rendering, programmatic SEO, and clean frontend architecture using real-world cryptocurrency data.
